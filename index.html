Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Système de consultation de pages de données permettante d'accéder aux ressources du réseau Internet

Internet

Ensemble de réseaux mondiaux interconnectés qui permet à des ordinateurs et à des serveurs de communiquer efficacement au moyen d'un protocole de communication commun (IP) et de services tels que le Web, le FTP, la messagerie ou les groupes de discussion

Cloud

Ensemble des serveurs distants proposant des services accessibles par le réseau Internet

W3C

Organisme international à but non lucratif, son rôle est de définir les standards techniques liés au web

Partie 2 - Distinguez les langages du Web

Langages client

Langages de programmation exécutés côté client (c'est-à-dire sur l'ordinateur de l'utilisateur) au lieu de côté serveur (sur un serveur distant)

Navigateurs

Logiciels clients pour l'affichage de pages Web au format HTML, qui permettent l'activation de liens hypertextes pour aller de site en site

Langages serveur

Langages de programmation exécutés côté serveur (c’est-à-dire sur un ordinateur distant qui héberge un site web ou une application) au lieu du côté client (sur l’ordinateur de l’utilisateur)

Frameworks

Ensemble cohérent de composants logiciels structurels qui sert à créer les fondations ainsi que les grandes lignes de tout ou partie d'un logiciel, c'est-à-dire une architecture

Bases de données

Systèmes électroniques permettant d'accéder facilement à un ensemble organisé de données, de les manipuler et de les mettre à jour

Site responsive

Site conçu et développé de façon à pouvoir s'adapter à toutes les résolutions d'écran (pas comme ilevia.fr)

Application mobile

Logiciel applicatif conçu pour un appareil électronique mobile, tel qu'un téléphone ou une tablette

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Systèmes informatiques fournissant des services, des ressources ou des données à d'autres ordinateurs, appelés clients, sur un réseau

Adresse IP

Identifiant unique de chaque machine connectée au réseau utilisant le protocole Internet

Nom d’hôte et DNS

Le nom d'hôte est le nom d'un périphérique dans un réseau ; Un DNS est un système permettant la conversion d'un nom de domaine vers son adresse IP, tel un annuaire téléphonique

Protocoles

Ensemble de règles régissant la manière dont les données sont échangées entre machines

Partie 4 - Identifiez les métiers du web

Développeurs

Traduisent les demandes du client en lignes de code ; créent et améliorent des logiciels, des applications, des sites web et autres systèmes informatiques

Product Managers et chefs de projets

Identifient les besoins des utilisateurs ; établissent la stratégie, la feuille de route et le cycle d'un produit ; supervisent la conception, le marketing, le design du produit

Designers

Conçoivent l'aspect visuel et l'interactivité des produits et services numériques, tels que les sites web, les applications ou le logiciels

Testeurs

Vérifient les performances d'un logiciel, d'un jeu vidéo ou d'une application mobile et signalent aux développeurs les anomalies détectées, le cas échéant

Techniciens informatique

Assurent l'installation, la maintenance, le dépannage et la sécurisation des systèmes informatiques d'une entreprise ou d'un organisme

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Langage décrivant la présentation d'un document web, notamment la mise en forme des éléments HTML

JavaScript

Front-End

Langage de programmation léger principalement utilisé pour créer des pages web dynamiques et interactives

JavaScript

Back-End

Utilisé entre autres pour gérer la logique du serveur, les interactions avec la base de données ou la création d'API

PHP

Back-End

Langage de script open source très utilisé dans le développement web permettant de créer des pages web dynamiques et souvent intégré avec HTML

Java

Back-End

Langage de programmation léger principalement utilisé pour créer des pages web dynamiques et interactives

Python

Back-End

Langage de programmation open source et polyvalent, utilisé dans divers domaines tels que le développement web, la science des données, l'apprentissage automatique ou l'automatisation

C#

Back-End

Orienté objet, conçu pour créer entre autres des applications de bureau, des applications web, des jeux ou des services web

SQL

Bases de données

Utilisé pour gérer et communiquer avec des bases de données, créer des requêtes et modifier ou récupérer des informations

Boîte à outils du développeur

  • IDE comme VS Code : application logicielle qui aide les programmeurs à développer efficacement le code logiciel
  • Github : plateforme open source de gestion de versions et de collaboration destinée aux développeurs de logiciels
  • Librairies / bibliothèques : entrepôt de code pré-construit utilisable par un public de programmeurs
  • Frameworks : ensemble de composants logiciels réutilisables qui permettent de développer de nouvelles applications plus efficacement (ex : Bootstrap - HTML, CSS et JavaScript ; Tailwind - CSS ; Svelte - JavaScript)
